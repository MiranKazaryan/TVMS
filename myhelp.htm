Руководство пользователя

Обучающая программа «TV and MS» предназначено для поддержки учебного процесса при изучении дисциплины «Теория вероятностей и математическая статистика» студентами технических специальностей.

Установка

В каталоге Install запустить setupEducationSystem.exe

После установки программы предлагается установить .Net Framework 2.0, Math Player. 

Руководство по использованию

Первоначально пользователю нужно пройти авторизацию или же регистрацию. 

В режиме "Регистрации" и "Авторизации" пользователю необходимо заполнить все поля и совершить вход в программу.

В программе разделенные права доступа для двух видов пользователей:

  • "Преподаватель"
  • "Студент"

При этом пользователь "Преподаватель" имеет возможность работы в 5 режимах

  • просмотр лекционного материала;
  • статистика;
  • тестирование;
  • добавление тестового набора;
  • редактирование тестовых заданий.

Пользователь "Студент" имеет возможность работы в 3 режимах

  • просмотр лекционного материала;
  • статистика;
  • тестирование;

Режим просмотра лекционного материала:

Открывается теоретический материал по дисциплине.

Режим статистики:

Открывается окно с собранной статистикой по пройденным тестированиям авторизованным пользователем.

Режим тестирования:

Подразделяется на два режима.

Обучающее тестирование:

В режиме обучения пользователю необходимо выбрать параметры тестирования: модули и параграфы из теоретического курса дисциплины,уровень сложности тестовых заданий.Количество решаемых заданий неограниченно и подается пользователю до тех пор, пока он не решит остановится.

Нажатием на кнопки «Следующее задание», «Проверить задание» последовательно прорешиваются задания.

При нажатии на кнопку «Завершение тестирования» осуществляется переход в окно статистика.

Контрольное тестирование:

Для работы в режиме контрольного тестирования предварительно необходимо выбрать тестовый набор из предложенных. Далее необходимо решить все тестовые задания и завершить тестирование.После этого можно просмотреть в каких заданиях были совершены ошибки.

Режим добавления тестового набора:

Для работы в режиме контрольного тестирования предварительно необходимо выбрать тестовый набор из предложенных.Добавить тестовый набор может только преподаватель. Необходимо выбрать параметры контрольного тестирования: модуль и параграфы из этого модуля, уровень сложности набора заданий, количество заданий (от 6 до 8) и название тестового набора.

Режим редактирования тестового задания:

Подразделяется на 3 режима.

Добавить, удалить и изменить тестовое задание может только преподаватель.

Добавление тестового задания:

Необходимо выбрать параметры тестового задания: модуль и параграфы,к которым относится тестовое задание, уровень сложности задания,тип тестового задания, ответ на тестовое задание и само тестовое задание.

Удаление тестового задания:

Необходимо выбрать тестовое задание из списка предложенных и при необходимости удалить его.

Изменить тестовое задание:

Необходимо выбрать тестовое задание из списка предложенных, при необходимости можно изменить само тестовое задание , или ответ на него.
